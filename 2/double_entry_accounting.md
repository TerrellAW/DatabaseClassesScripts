## Double Entry Accounting

### Accounting

#### Types of Accounting

1. Single-entry accounting:
    - Keeps track of cash
2. Double-entry accounting:
    - Tracks cash as well as its origin and destination

#### Terminology

- Period: 1 month

#### Account Categories

1. Assets (A):
    - Default transaction type = Debit
    - Anything that retains value for a long period of time
        - Cash
        - Buildings
        - Investments
        - Furniture
        - Receivables - what is owed to company
2. Liabilities (L):
    - Default transaction type = Credit
    - Debt
        - Mortgage
        - Loans
        - Payables - what the company owes
3. Owner's Equity (OE):
    - Default transaction type = Credit
    - For sole-proprietorship
        - Corporations have Shareholder Equity
        - Partnerships have Partner's Equity
    - Cumulative profit/loss since start of the company
        - OE = OE + RE - EX each period
4. Revenue (RE):
    - Default transaction type = Credit
    - Temporary account for one period
        - Added into OE and then reset after one period
    - Income of the company
        - From selling products/services
5. Expenses (EX):
    - Default transaction type = Debit
    - Temporary account for one period
        - Subtracted into OE and then reset after one period
    - Routine expenditures of the company
        - Expenses on things that do not retain value
        - Bills
        - Consumables

### Double Entry Accounting Transactions

A double entry account transaction is made up of atleast two rows of data.

#### Rules for Transactions

1. Within a transaction Debits (D) = Credits (C).
2. Transaction type can be D or C
    - Every account has a default transaction type
3. Each row will be given an account and a transaction type
    - Get default transaction type for account
    - Compare to default transaction type in the row:
        - If same, `ADD`
        - If different, `SUBTRACT`

#### Accounting Equation

``` math
A = L + OE
```

``` math
A = L + OE + RE - EX
```

### Examples

Transaction using accounting notation:

```
Dr: Office Supply Expense      200
  Cr: Cash                       150
  Cr: Office Supply Payables      50
 Bought office supplies
```

Debits = Credits, so it is valid.

When a transaction is not its default type it is negative, else it is positive.
In this case office supply expense is increasing, cash is decreasing and office supply payables is increasing.

``` math
A(Decreasing) = L(Increasing) + OE(Decreasing) + RE - EX(Increasing)
```
